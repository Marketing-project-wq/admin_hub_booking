## IMPLEMENTATION DETAIL

### `src/lib/supabase.js`

```javascript
import { createClient } from '@supabase/supabase-js'

const url = import.meta.env.VITE_SUPABASE_URL
const key = import.meta.env.VITE_SUPABASE_ANON_KEY

if (!url || !key) {
  console.error('Missing Supabase env vars')
}

export const supabase = createClient(url, key)
```

### `src/context/AuthContext.jsx`

- State: `user` (object: `{id, email, full_name, role}` atau `null`), `loading`
- `login(email, password)`: call RPC `validate_admin_login`, simpan ke `localStorage` key `admin_user`, set state
- `logout()`: clear `localStorage`, set state ke `null`
- Saat mount: read `localStorage` untuk restore session (no token refresh needed, ini simple session)
- Provide via `<AuthContext.Provider>` di `App.jsx`
- Hook: `useAuth()`

```javascript
const login = async (email, password) => {
  const { data, error } = await supabase.rpc('validate_admin_login', {
    p_email: email,
    p_password: password,
  })
  if (error) throw error
  if (!data || data.length === 0) {
    throw new Error('Email atau password salah')
  }
  const profile = data[0]
  localStorage.setItem('admin_user', JSON.stringify(profile))
  setUser(profile)
  return profile
}
```

### `src/context/UnitContext.jsx`

- State: `selectedUnit` — string `'arena' | 'gym' | 'clinic' | null`
- `selectUnit(unit)`: set state + write `localStorage` key `selected_unit`
- `clearUnit()`: clear keduanya (untuk "Switch Unit" balik ke landing)
- Restore dari localStorage saat mount
- Hook: `useUnit()`

### `src/components/ProtectedRoute.jsx`

```jsx
const ProtectedRoute = ({ children, requireUnit = false }) => {
  const { user, loading } = useAuth()
  const { selectedUnit } = useUnit()

  if (loading) return <div className="loading-screen">Loading...</div>
  if (!user) return <Navigate to="/login" replace />
  if (requireUnit && !selectedUnit) return <Navigate to="/select" replace />

  return children
}
```

### `src/App.jsx` — Routing

```jsx
<BrowserRouter>
  <AuthProvider>
    <UnitProvider>
      <Routes>
        <Route path="/login" element={<LoginPage />} />
        <Route path="/select" element={
          <ProtectedRoute><SelectUnitPage /></ProtectedRoute>
        } />
        <Route element={<ProtectedRoute requireUnit><Layout /></ProtectedRoute>}>
          <Route path="/arena" element={<ArenaDashboard />} />
          <Route path="/gym" element={<GymDashboard />} />
          <Route path="/clinic" element={<ClinicComingSoon />} />
        </Route>
        <Route path="*" element={<Navigate to="/login" replace />} />
      </Routes>
    </UnitProvider>
  </AuthProvider>
</BrowserRouter>
```

`<Layout />` pakai `<Outlet />` untuk nested route.
