## LOGIKA KALKULASI

### Venue Occupancy Rate
```
Slot tersedia per hari = 14 jam
Total slot tersedia = 14 × jumlah hari dalam periode
Slot terpakai = SUM(LEAST(end_time - start_time, 14 jam)) per booking confirmed
  → cap 14 jam per hari (tidak bisa > 100%)
Occupancy Rate = slot terpakai / total slot tersedia × 100%
```

### Class Occupancy Rate
```
Total kapasitas = COUNT(active schedules dalam periode) × 40 peserta
Peserta masuk = COUNT(confirmed class bookings dalam periode)
Occupancy Rate = peserta masuk / total kapasitas × 100%
```

## DATA FETCHING

### Fetch Venue Occupancy

```javascript
const fetchVenueOccupancy = async (start, end) => {
  // Hitung jumlah hari dalam periode
  const startDate = new Date(start)
  const endDate   = new Date(end)
  const days = Math.ceil((endDate - startDate) / (1000 * 60 * 60 * 24))
  const totalSlots = days * 14 // 14 jam/hari

  // Fetch semua venue booking confirmed dalam periode
  const { data } = await supabase
    .from('arena_bookings')
    .select('booking_date, start_time, end_time')
    .eq('status', 'confirmed')
    .gte('booking_date', start.slice(0, 10))
    .lte('booking_date', end.slice(0, 10))

  if (!data || data.length === 0) {
    setVenueOccupancy(0)
    return
  }

  // Hitung jam terpakai per hari, cap 14 jam
  const hoursPerDay = {}
  data.forEach(b => {
    const date = b.booking_date
    const startH = parseInt(b.start_time?.slice(0, 2) || '0')
    const startM = parseInt(b.start_time?.slice(3, 5) || '0')
    const endH   = parseInt(b.end_time?.slice(0, 2) || '0')
    const endM   = parseInt(b.end_time?.slice(3, 5) || '0')
    const duration = ((endH * 60 + endM) - (startH * 60 + startM)) / 60
    hoursPerDay[date] = (hoursPerDay[date] || 0) + Math.max(0, duration)
  })

  // Sum jam terpakai, cap 14 per hari
  const totalUsed = Object.values(hoursPerDay)
    .reduce((sum, h) => sum + Math.min(h, 14), 0)

  const rate = totalSlots > 0 ? Math.round((totalUsed / totalSlots) * 100) : 0
  setVenueOccupancy(Math.min(rate, 100)) // cap 100%
}
```

### Fetch Class Occupancy

```javascript
const fetchClassOccupancy = async (start, end) => {
  const CAPACITY_PER_CLASS = 40

  const [schedulesRes, bookingsRes] = await Promise.all([
    // Hitung jadwal aktif dalam periode
    supabase
      .from('arena_class_schedules')
      .select('id', { count: 'exact', head: true })
      .eq('is_cancelled', false)
      .gte('schedule_date', start.slice(0, 10))
      .lte('schedule_date', end.slice(0, 10)),

    // Hitung peserta confirmed dalam periode
    supabase
      .from('arena_class_bookings')
      .select('id', { count: 'exact', head: true })
      .eq('status', 'confirmed')
      .gte('paid_at', start)
      .lte('paid_at', end),
  ])

  const totalCapacity = (schedulesRes.count || 0) * CAPACITY_PER_CLASS
  const totalPeserta  = bookingsRes.count || 0
  const rate = totalCapacity > 0 ? Math.round((totalPeserta / totalCapacity) * 100) : 0

  setClassOccupancy(Math.min(rate, 100))
}
```

## UI — KOMPONEN GAUGE/DONUT

Tambah komponen `OccupancyGauge` di file yang sama:

```jsx
function OccupancyGauge({ value, label, color }) {
  const radius = 36
  const circumference = 2 * Math.PI * radius
  const offset = circumference - (value / 100) * circumference

  return (
    <div style={{
      display: 'flex', flexDirection: 'column', alignItems: 'center',
      justifyContent: 'center', gap: 8,
    }}>
      <div style={{ position: 'relative', width: 90, height: 90 }}>
        <svg width="90" height="90" style={{ transform: 'rotate(-90deg)' }}>
          {/* Background circle */}
          <circle
            cx="45" cy="45" r={radius}
            fill="none"
            stroke="#F3F4F6"
            strokeWidth="8"
          />
          {/* Progress circle */}
          <circle
            cx="45" cy="45" r={radius}
            fill="none"
            stroke={color}
            strokeWidth="8"
            strokeDasharray={circumference}
            strokeDashoffset={offset}
            strokeLinecap="round"
            style={{ transition: 'stroke-dashoffset 0.6s ease' }}
          />
        </svg>
        {/* Value text di tengah */}
        <div style={{
          position: 'absolute', inset: 0,
          display: 'flex', alignItems: 'center', justifyContent: 'center',
          fontSize: 16, fontWeight: 700, color: 'var(--text-primary)',
        }}>
          {value}%
        </div>
      </div>
      <div style={{
        fontSize: 12, fontWeight: 600, color: 'var(--text-muted)',
        textAlign: 'center', lineHeight: 1.3,
      }}>
        {label}
      </div>
    </div>
  )
}
```
