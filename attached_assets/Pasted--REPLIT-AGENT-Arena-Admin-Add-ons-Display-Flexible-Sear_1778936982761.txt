### 1c. Tampilkan add-ons di Detail Modal

Di `BookingDetailModal` atau detail view, tambahkan section add-ons setelah section harga:

```jsx
{/* Section Add-ons */}
{booking.addons?.length > 0 && (
  <div className="modal-section">
    <div style={{ fontSize: 11, color: 'var(--text-muted)', textTransform: 'uppercase', letterSpacing: '0.05em', marginBottom: 8 }}>
      Add-ons
    </div>
    {booking.addons.map(a => (
      <div key={a.id} className="detail-row">
        <span className="detail-label">{a.addon_name} ×{a.qty}</span>
        <span className="detail-value">{fmtRp(a.subtotal)}</span>
      </div>

      <span className="detail-label" style={{ fontWeight: 600 }}>Total Addon</span>
      <span className="detail-value" style={{ fontWeight: 600 }}>
        {fmtRp(booking.addons.reduce((s, a) => s + a.subtotal, 0))}
      </span>
    </div>
    <div className="detail-row">
      <span className="detail-label" style={{ fontWeight: 700 }}>Grand Total</span>
      <span className="detail-value" style={{ fontWeight: 700, color: 'var(--red)' }}>
        {fmtRp(booking.price + booking.addons.reduce((s, a) => s + a.subtotal, 0))}
      </span>
    </div>
  </div>
)}
```

## UPDATE 2 — FLEXIBLE SEARCH DI SEMUA MENU

Untuk setiap halaman, ganti search yang hanya cari 1-2 field menjadi search yang lebih luas.

### ArenaClassBookings.jsx

**Search fields:** booking_code, full_name, email, phone, group_id (4 char pertama), payment_method

```javascript
// Update query search:
if (search) {
  // Cek apakah search match group_id prefix (4 char hex)
  const isGroupSearch = /^[0-9a-f]{4}$/i.test(search.trim())

  if (isGroupSearch) {
    // Search by group_id prefix — fetch semua lalu filter client-side
    // karena Supabase tidak support LIKE pada UUID langsung
    query = query.or(`full_name.ilike.%${search}%,booking_code.ilike.%${search}%,email.ilike.%${search}%,phone.ilike.%${search}%`)
    // group_id filter ditambahkan setelah fetch via post-processing
  } else {
    query = query.or(
      `full_name.ilike.%${search}%,` +
      `booking_code.ilike.%${search}%,` +
      `email.ilike.%${search}%,` +
      `phone.ilike.%${search}%,` +
      `payment_method.ilike.%${search}%`
    )
  }
}

// Post-processing untuk group_id search:
const isGroupSearch = /^[0-9a-f]{4}$/i.test(search.trim())
if (isGroupSearch && rows) {
  const filtered = rows.filter(r =>
    r.group_id?.startsWith(search.toLowerCase()) ||
    r.full_name?.toLowerCase().includes(search.toLowerCase()) ||
    r.booking_code?.toLowerCase().includes(search.toLowerCase()) ||
    r.email?.toLowerCase().includes(search.toLowerCase()) ||
    r.phone?.includes(search)
  )
  setData(filtered)
} else {
  setData(rows || [])
}
```
