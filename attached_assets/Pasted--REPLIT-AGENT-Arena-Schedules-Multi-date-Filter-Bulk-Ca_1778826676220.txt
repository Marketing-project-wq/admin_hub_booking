# REPLIT AGENT — Arena Schedules: Multi-date Filter + Bulk Cancel

> **Context:** Admin Arena panel, halaman Schedules (`/arena/master/schedules`).
> **Tujuan:** Tambah 2 fitur:
> 1. Filter multi-tanggal (pilih beberapa tanggal sekaligus)
> 2. Bulk select + cancel beberapa jadwal sekaligus
>
> **JANGAN ubah:** logika fetch, join, booked count, atau fitur lain yang sudah ada.

---

## FITUR 1 — FILTER MULTI-TANGGAL

### Behavior
- Ganti date picker single menjadi **date range** (dari tanggal - sampai tanggal)
- Tambah tombol **"Semua Tanggal"** untuk reset filter
- Default: tampilkan dari hari ini ke depan (7 hari)

### UI

```jsx
<div className="filter-bar">
  {/* Date range */}
  <div style={{ display: 'flex', alignItems: 'center', gap: 8 }}>
    <input
      type="date"
      value={dateFrom}
      onChange={e => { setDateFrom(e.target.value); setPage(0) }}
    />
    <span style={{ color: 'var(--text-muted)', fontSize: 13 }}>s/d</span>
    <input
      type="date"
      value={dateTo}
      onChange={e => { setDateTo(e.target.value); setPage(0) }}
    />
  </div>

  {/* Class type filter (existing) */}
  <select value={classTypeFilter} onChange={e => setClassTypeFilter(e.target.value)}>
    <option value="all">Semua Class Type</option>
    {classTypes.map(ct => (
      <option key={ct.id} value={ct.id}>{ct.name}</option>
    ))}
  </select>

  {/* Status filter */}
  <select value={statusFilter} onChange={e => setStatusFilter(e.target.value)}>
    <option value="active">Aktif</option>
    <option value="cancelled">Dibatalkan</option>
    <option value="all">Semua Status</option>
  </select>

  {/* Reset */}
  <button
    className="btn-secondary"
    style={{ fontSize: 12, padding: '6px 12px' }}
    onClick={() => {
      const today = new Date().toISOString().slice(0, 10)
      const next7 = new Date(Date.now() + 7*24*60*60*1000).toISOString().slice(0, 10)
      setDateFrom(today)
      setDateTo(next7)
      setClassTypeFilter('all')
      setStatusFilter('active')
      setPage(0)
    }}
  >
    Reset
  </button>
</div>
```

### State baru
```javascript
const today = new Date().toISOString().slice(0, 10)
const next7  = new Date(Date.now() + 7*24*60*60*1000).toISOString().slice(0, 10)

const [dateFrom, setDateFrom] = useState(today)
const [dateTo, setDateTo] = useState(next7)
const [statusFilter, setStatusFilter] = useState('active')
```

### Fetch query update
```javascript
let query = supabase
  .from('arena_class_schedules')
  .select(`...existing select...`, { count: 'exact' })
  .gte('schedule_date', dateFrom)
  .lte('schedule_date', dateTo)
  .order('schedule_date', { ascending: true })
  .order('start_time', { ascending: true })

if (classTypeFilter !== 'all') query = query.eq('class_type_id', classTypeFilter)
if (statusFilter === 'active') query = query.eq('is_cancelled', false)
if (statusFilter === 'cancelled') query = query.eq('is_cancelled', true)
```

## FITUR 2 — BULK SELECT + CANCEL

### Behavior
- Checkbox di setiap row untuk select
- Checkbox di header untuk select all (di page saat ini)
- Floating action bar muncul di bawah saat ada yang di-select
- Action bar: "X jadwal dipilih" + tombol "Cancel Semua" + tombol "Batal Pilih"
- Konfirmasi sebelum bulk cancel
- Setelah cancel: deselect semua, refetch

### State
```javascript
const [selected, setSelected] = useState<Set<string>>(new Set())
const [showBulkConfirm, setShowBulkConfirm] = useState(false)
const [bulkLoading, setBulkLoading] = useState(false)
```

### Checkbox header (select all di page ini)
```jsx
<th style={{ width: 40 }}>
  <input
    type="checkbox"
    checked={data.filter(r => !r.is_cancelled).length > 0 &&
             data.filter(r => !r.is_cancelled).every(r => selected.has(r.id))}
    onChange={e => {
      const activIds = data.filter(r => !r.is_cancelled).map(r => r.id)
      if (e.target.checked) {
        setSelected(prev => new Set([...prev, ...activIds]))
      } else {
        setSelected(prev => {
          const next = new Set(prev)
          activIds.forEach(id => next.delete(id))
          return next
        })
      }
    }}
  />
</th>
```

### Checkbox per row
```jsx
<td>
  {!row.is_cancelled && (
    <input
      type="checkbox"
      checked={selected.has(row.id)}
      onChange={e => {
        setSelected(prev => {
          const next = new Set(prev)
          e.target.checked ? next.add(row.id) : next.delete(row.id)
          return next
        })
      }}
    />
  )}
</td>
```

### Floating action bar
Muncul fixed di bottom saat `selected.size > 0`:

```jsx
{selected.size > 0 && (
  <div style={{
    position: 'fixed',
    bottom: 32,
    left: '50%',
    transform: 'translateX(-50%)',

    color: '#FFFFFF',

  }}>

    </span>
    <button

      style={{ padding: '8px 16px', fontSize: 13 }}
      onClick={() => setShowBulkConfirm(true)}
    >
      Cancel Semua
    </button>
    <button
      style={{
        background: 'transparent',
        border: '1px solid rgba(255,255,255,0.3)',
        color: '#FFFFFF',
        borderRadius: 4,
        padding: '8px 16px',
        fontSize: 13,
        cursor: 'pointer',
        fontFamily: 'inherit',
      }}
      onClick={() => setSelected(new Set())}
    >
      Batal Pilih
    </button>
  </div>
)}
```

### Bulk cancel handler
```javascript
const handleBulkCancel = async () => {
  setBulkLoading(true)
  try {
    const ids = Array.from(selected)
    const { error } = await supabase
      .from('arena_class_schedules')
      .update({
        is_cancelled: true,
        cancelled_reason: 'Dibatalkan oleh admin',
        updated_at: new Date().toISOString(),
      })
      .in('id', ids)

    if (error) throw error

    setSelected(new Set())
    setShowBulkConfirm(false)
    fetchData()
  } catch (err) {
    console.error('Bulk cancel error:', err)
  } finally {
    setBulkLoading(false)
  }
}
```

### Confirm modal untuk bulk cancel
```jsx
{showBulkConfirm && (
  <ConfirmModal
    title="Cancel Jadwal"
    message={`Batalkan ${selected.size} jadwal yang dipilih? Tindakan ini tidak bisa dibatalkan.`}
    onConfirm={handleBulkCancel}
    onCancel={() => setShowBulkConfirm(false)}
    danger
  />
)}
```

### Reset selected saat page/filter berubah
```javascript
useEffect(() => {
  setSelected(new Set())
}, [dateFrom, dateTo, classTypeFilter, statusFilter, page])
```
