# REPLIT AGENT — Arena Admin: Tambah Menu "User Management"

> **Context:** Admin Panel Arena. Menu ini hanya visible untuk role `super_admin`.
> **Tujuan:** Super admin bisa manage akun admin/staff Arena — tambah, edit, reset password, toggle active, set permissions.
>
> **Schema:** tabel `admin_users` + RPC functions yang sudah ada di Supabase.
> **JANGAN ubah:** menu lain, auth flow, routing lain.

## SCHEMA REFERENCE

### Tabel `admin_users`
```
id UUID, email TEXT, full_name TEXT, role TEXT, unit TEXT,
password_hash TEXT, permissions JSONB, is_active BOOLEAN,
last_login_at TIMESTAMPTZ, created_at TIMESTAMPTZ, updated_at TIMESTAMPTZ
```

**PENTING:** Karena RLS `admin_users` block direct SELECT, perlu buat RPC baru dulu.
Sebelum paste prompt ini, jalankan SQL migration berikut di Supabase:

```sql
-- RPC untuk list users per unit (hanya arena untuk sekarang)
CREATE OR REPLACE FUNCTION get_admin_users(p_unit TEXT DEFAULT NULL)
RETURNS TABLE(
  id UUID, email TEXT, full_name TEXT, role TEXT, unit TEXT,
  permissions JSONB, is_active BOOLEAN, last_login_at TIMESTAMPTZ, created_at TIMESTAMPTZ
)
LANGUAGE plpgsql
SECURITY DEFINER
SET search_path = public, extensions
AS $$
BEGIN
  RETURN QUERY
  SELECT u.id, u.email, u.full_name, u.role, u.unit,
         u.permissions, u.is_active, u.last_login_at, u.created_at
    FROM admin_users u
   WHERE (p_unit IS NULL OR u.unit = p_unit OR u.role = 'super_admin')
   ORDER BY u.role, u.created_at;
END;
$$;

GRANT EXECUTE ON FUNCTION get_admin_users(TEXT) TO anon, authenticated;

-- RPC untuk update user info (non-password fields)
CREATE OR REPLACE FUNCTION update_admin_user(
  p_user_id   UUID,
  p_full_name TEXT,
  p_role      TEXT,
  p_is_active BOOLEAN
) RETURNS BOOLEAN
LANGUAGE plpgsql
SECURITY DEFINER
SET search_path = public, extensions
AS $$
BEGIN
  UPDATE admin_users
     SET full_name = p_full_name,
         role = p_role,
         is_active = p_is_active,
         updated_at = NOW()
   WHERE id = p_user_id;
  RETURN FOUND;
END;
$$;

GRANT EXECUTE ON FUNCTION update_admin_user(UUID, TEXT, TEXT, BOOLEAN) TO authenticated;
```

## MODAL 1: Tambah User

```jsx
// Fields:
// - Nama Lengkap (text, required)
// - Email (email, required)
// - Role: admin / staff (radio)
// - Password (text, required, min 8 char)
// - Konfirmasi Password (text, harus match)

const handleAddUser = async () => {
  if (form.password !== form.confirmPassword) {
    setFormError('Password tidak cocok')
    return
  }
  if (form.password.length < 8) {
    setFormError('Password minimal 8 karakter')
    return
  }
  setSubmitting(true)
  const { error } = await supabase.rpc('create_admin_user', {
    p_email:       form.email.toLowerCase().trim(),
    p_full_name:   form.fullName.trim(),
    p_role:        form.role,
    p_unit:        'arena',
    p_password:    form.password,
    p_permissions: {},
  })
  if (error) {
    setFormError(error.message.includes('duplicate') ? 'Email sudah terdaftar' : error.message)
  } else {
    setShowAddModal(false)
    setSuccess(`User ${form.email} berhasil dibuat`)
    fetchUsers()
  }
  setSubmitting(false)
}
```

## MODAL 4: Set Permissions (hanya untuk role staff)

Tampilkan checklist semua permission keys:

```jsx
const PERMISSION_KEYS = [
  { key: 'dashboard.view',  label: 'Lihat Dashboard' },
  { key: 'bookings.view',   label: 'Lihat Bookings' },
  { key: 'bookings.edit',   label: 'Edit / Confirm / Cancel Booking' },
  { key: 'bookings.create', label: 'Buat Manual Booking' },
  { key: 'venue.view',      label: 'Lihat Venue Booking' },
  { key: 'venue.edit',      label: 'Edit Venue Booking' },
  { key: 'vouchers.view',   label: 'Lihat Vouchers' },
  { key: 'vouchers.edit',   label: 'Buat / Edit Voucher' },
  { key: 'packages.view',   label: 'Lihat Package Orders' },
  { key: 'schedules.view',  label: 'Lihat Schedules' },
  { key: 'schedules.edit',  label: 'Tambah / Edit / Cancel Schedule' },
  { key: 'master.view',     label: 'Lihat Master Data' },
  { key: 'master.edit',     label: 'Edit Master Data' },
]

// Init dari existing permissions
const [perms, setPerms] = useState({})
useEffect(() => {
  if (permUser) setPerms(permUser.permissions || {})
}, [permUser])

const togglePerm = (key) => setPerms(p => ({ ...p, [key]: !p[key] }))

// "Select All" shortcut
const selectAll = () => {
  const all = {}
  PERMISSION_KEYS.forEach(p => { all[p.key] = true })
  setPerms(all)
}
const clearAll = () => setPerms({})

const handleSavePermissions = async () => {
  setSubmitting(true)
  const { error } = await supabase.rpc('update_admin_permissions', {
    p_user_id:     permUser.id,
    p_permissions: perms,
  })
  if (error) {
    setFormError(error.message)
  } else {
    setPermUser(null)
    setSuccess(`Permissions ${permUser.full_name} berhasil disimpan`)
    fetchUsers()
  }
  setSubmitting(false)
}
```

**UI checklist:**
```jsx
<div style={{ display: 'flex', justifyContent: 'flex-end', gap: 8, marginBottom: 12 }}>

  <button className="btn-secondary" style={{ fontSize: 12, padding: '4px 10px' }}
    onClick={selectAll}>Select All</button>
</div>

{PERMISSION_KEYS.map(p => (
```
